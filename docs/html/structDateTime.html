Detailed Description

Represents any date or time with optional fields.

String Format: YYYYmmDDHHMMSS

  • DateTime structure is generated from an array of 14 characters where each can be a number or '?'. If any date time field starts with a '?' that section is ignored.

  • After the sturcture is generated DT_WC enum can be used on DateTime.flag to obtain ignored fields.

    Example:

    • 202112201830?? 20 December 2012 18:30
    • ????????123015 12:30:15
